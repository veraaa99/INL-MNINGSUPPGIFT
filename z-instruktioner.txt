Kriterier för att få godkänt på denna inlämningsuppgift :

- Du behöver endast göra designen för desktop, dvs. sidan behöver inte vara responsiv.
- Du får välja mellan att göra det ljus eller det mörka temat.
- Alla delar på sidan ska göras.
- Du behöver inte göra det möjligt att kunna växla mellan mörkt och ljust tema.
- Ditt projekt ska versionshanteras med Git och vara publicerat på GITHUB, du behöver inte använda dig av branches.

Kriterier för att få väl godkänt på denna inlämningsuppgift :

- Kraven för godkänt måste vara uppfyllda.
- Du ska göra designen för både desktop, tablet och mobile. Dvs. sidan ska vara responsiv och ändras med hjälp av media queries och/eller container queries.
- Du ska använda dig av custom properties (variabler) för att hantera alla färger.
- Du ska kunna sätta på / stänga av "dark mode" med hjälp av en klass på html elementet.
- Du ska använda ett färdigt JavaScript (som tillhandahålls av läraren) för att sköta växlingen mellan mörkt och ljust tema.
